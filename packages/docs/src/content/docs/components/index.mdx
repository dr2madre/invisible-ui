Invisible UI ships **headless primitives** in the framework-agnostic core, with a
**styled, themeable Svelte layer** on top. Every component is accessible by
default (WAI-ARIA patterns, keyboard support, axe-tested) and themeable through
`--ds-*` custom properties.

Pick a component from the sidebar to see live demos, prop tables, and usage.

## Categories

- **Actions** — Button, Button Group, Toggle Button, Toolbar.
- **Forms & values** — Text Field, Select, Combobox, Checkbox, Radio Group,
  Switch, Slider, Segmented Control, Toggle Group, PIN Input, Date & Time
  pickers.
- **Overlays** — Dialog, Alert Dialog, Sheet, Drawer, Popover, Tooltip,
  Hover Card, Dropdown / Context Menu, Command, Navigation Menu.
- **Feedback** — Alert, Notice, Progress, Meter, Skeleton, Feedback Icon.
- **Data & presentation** — Table, Tree View, Avatar (+ Group), Card, Tag,
  Badge / Count, Carousel, Stepper, Pagination, Code & Code Block.
