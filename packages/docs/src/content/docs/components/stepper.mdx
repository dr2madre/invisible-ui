An ordered progress indicator (timeline / steps) with linear gating and
`aria-current` on the active step.

### Import

```svelte
<script>
  import Stepper from "@design-system/svelte/Stepper.svelte";
</script>
```

### Props

<PropsTable component="stepper" />

### Accessibility

- The active step carries `aria-current="step"`; completed/upcoming steps are
  conveyed for assistive tech. In `linear` mode, locked steps aren't actionable.

### Example

```svelte
<Stepper current={1} steps={[{ label: "Cart" }, { label: "Pay" }]} />
```

Themeable via `--ds-stepper-*`.
