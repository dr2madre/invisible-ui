Sets the locale, writing **direction**, and **message overrides** for everything
inside it. Components read their default labels from this context, so you
localize once instead of passing a prop to every component; `dir` flips the CSS
logical properties for RTL.

### Import

```svelte
<script>
  import LocaleProvider from "@design-system/svelte/LocaleProvider.svelte";
</script>
```

### Props

<PropsTable component="locale-provider" />

Any label prop you pass to a component still wins over the catalog. Default
strings live in the English catalog (`en`); see `MessageKey` for the keys.

### Example

```svelte
<LocaleProvider locale="it" messages={{ "calendar.today": "Oggi" }}>
  <Calendar />
</LocaleProvider>
```
