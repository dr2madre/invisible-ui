A styled form label associated with a control via `for`, with an optional
required marker.

### Import

```svelte
<script>
  import Label from "@design-system/svelte/Label.svelte";
</script>
```

### Props

<PropsTable component="label" />

The label text is the default slot.

### Example

```svelte
<Label for="email" required>Email</Label>
<input id="email" type="email" />
```

Themeable via `--ds-label-*`.
