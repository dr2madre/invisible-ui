An independent on/off control (e.g. Bold in a toolbar), built on a native
`<input type="checkbox">` styled to look like a button.

### Import

```svelte
<script>
  import ToggleButton from "@design-system/svelte/ToggleButton.svelte";
</script>
```

### Props

<PropsTable component="toggle-button" />

### Accessibility

- A native checkbox: the browser owns the role, Space activation, focus and
  form participation. Exposes `data-state="on" \| "off"` for styling.
- Provide `label` when the slot content is icon-only so the control still has an accessible name.
- Pass `name` (and optional `value`) to submit the pressed state with a form.
- Distinct from `Switch`: a toggle button is an action, a switch is a setting.

### Example

```svelte
<ToggleButton label="Bold" onPressedChange={(on) => setBold(on)}>B</ToggleButton>
```

Themeable via `--ds-toggle-*`.
