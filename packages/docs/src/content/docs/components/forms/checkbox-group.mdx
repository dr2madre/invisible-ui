A multi-select group of checkboxes — a native `<fieldset>` of
`<input type="checkbox">` items.

### Import

```svelte
<script>
  import CheckboxGroup from "@design-system/svelte/CheckboxGroup.svelte";
</script>
```

### Props

<PropsTable component="checkbox-group" />

### Accessibility

- A native `<fieldset>`/`<legend>` group of `<input type="checkbox">`: each item is
  Tab-focusable and toggled with Space, and several can be checked at once.
- Supports per-item and whole-group `disabled` (the `<fieldset disabled>` cascades);
  the group needs an accessible `label`.

### Example

```svelte
<CheckboxGroup
  label="Toppings"
  value={["cheese"]}
  items={[
    { value: "cheese", label: "Cheese" },
    { value: "olives", label: "Olives" },
    { value: "anchovies", label: "Anchovies", disabled: true },
  ]}
  onValueChange={(v) => (selected = v)}
/>
```

Themeable via `--ds-checkbox-*`.
