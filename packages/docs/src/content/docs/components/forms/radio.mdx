A single radio button on a native `<input type="radio">`, grouped by `name` —
for hand-laid-out radios. For a managed set, use
[Radio Group](/components/radio-group/).

### Import

```svelte
<script>
  import Radio from "@design-system/svelte/Radio.svelte";
</script>
```

### Props

<PropsTable component="radio" />

### Accessibility

- A native `<input type="radio">`, so grouping and keyboard behaviour are the
  platform's.

### Example

```svelte
<Radio name="plan" value="pro" label="Pro" checked />
```

Themeable via `--ds-radio-*`.
