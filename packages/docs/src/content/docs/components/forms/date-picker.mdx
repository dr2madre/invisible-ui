A date field that opens a [Calendar](/components/calendar/) in a popover. The
readonly combobox field is the trigger; picking a day fills the field and
closes the popover.

### Import

```svelte
<script>
  import DatePicker from "@design-system/svelte/DatePicker.svelte";
</script>
```

### Props

<PropsTable component="date-picker" />

### Accessibility

- The field is a readonly `role="combobox"` (`aria-haspopup`, `aria-expanded`):
  click / Enter / Space opens; focus moves into the calendar; Escape returns
  focus to the field.
- See the [Calendar](/components/calendar/) for the grid's keyboard model.

### Example

```svelte
<DatePicker value="2026-06-15" label="Event date" clearable />
```

Themeable via `--ds-date-picker-*` and the calendar's `--ds-calendar-*`.
