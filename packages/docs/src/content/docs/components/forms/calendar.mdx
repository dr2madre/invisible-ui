A WAI-ARIA date grid with keyboard navigation, selection, appointment dots and
per-day prices. Switch between month, two-month, week, three-day, day and year
views.

### Import

```svelte
<script>
  import Calendar from "@design-system/svelte/Calendar.svelte";
</script>
```

### Props

<PropsTable component="calendar" />

### Accessibility

- `role="grid"` of day buttons with roving tabindex; arrows move by day/week,
  Home/End to the week edges, PageUp/PageDown by month (Shift = year),
  Enter/Space selects.
- The period title is announced (`aria-live`); out-of-range days are
  `aria-disabled`; today is `aria-current="date"`.

### Example

```svelte
<Calendar
  value="2026-06-15"
  views={["month", "week", "day"]}
  events={[{ date: "2026-06-18", label: "Review", tone: "warning" }]}
  prices={{ "2026-06-12": "€120" }}
/>
```

Themeable via `--ds-calendar-*`.
