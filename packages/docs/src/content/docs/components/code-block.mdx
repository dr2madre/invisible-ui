A block `<pre><code>` with an optional language caption and a copy button.

### Import

```svelte
<script>
  import CodeBlock from "@design-system/svelte/CodeBlock.svelte";
</script>
```

### Props

<PropsTable component="code-block" />

### Accessibility

- The copy button is labelled and announces success; the code is selectable.

### Example

```svelte
<CodeBlock code={"const x = 1;"} language="ts" />
```

Themeable via `--ds-code-block-*`.
