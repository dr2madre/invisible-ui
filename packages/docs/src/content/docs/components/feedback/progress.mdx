A `role="progressbar"` for task completion. Pass `value={null}` for an
indeterminate bar.

The demo also gathers the progress-type loaders from
[Loading Spinners](/invisible-ui/components/feedback/loading/) (`variant="bar"`):
label + percentage, the double bar for multi-file downloads, and the bar at the
top of a card — indeterminate and determinate.

### Import

```svelte
<script>
  import Progress from "@design-system/svelte/Progress.svelte";
</script>
```

### Props

<PropsTable component="progress" />

### Accessibility

- `role="progressbar"` with `aria-valuenow`/`min`/`max` (omitted when
  indeterminate). For a measurement (not completion) use [Meter](/components/meter/).

### Example

```svelte
<Progress value={62} label="Uploading" />
```

Themeable via `--ds-progress-*`.
