A modal that interrupts the user to confirm a consequential action (WAI-ARIA
alertdialog pattern). Reuses the headless dialog with `role="alertdialog"` and
the shared modal adapter. It has no close (✕) button; a backdrop press cancels
by default — set `closeOnOutsideClick={false}` to force an explicit choice for
irreversible actions.

The two examples above share their wording with the sibling presets so the
family reads consistently: the plain yes/no is the same choice as
[Confirm Dialog](/invisible-ui/components/overlays/confirm-dialog/), and the
type-to-confirm variant is the same input as
[Prompt Dialog](/invisible-ui/components/overlays/prompt-dialog/). The difference
is the role and urgency — AlertDialog interrupts (`role="alertdialog"`), the
presets do not (`role="dialog"`).

### When to use which dialog

- **[Dialog](/invisible-ui/components/overlays/dialog/)** — generic modal for any content.
- **[Confirm Dialog](/invisible-ui/components/overlays/confirm-dialog/)** — a routine yes/no confirmation.
- **Alert Dialog** — an *urgent* or *destructive* action to acknowledge (optional type-to-confirm).
- **[Prompt Dialog](/invisible-ui/components/overlays/prompt-dialog/)** — asks for a single value.

### Import

```svelte
<script>
  import AlertDialog from "@design-system/svelte/AlertDialog.svelte";
</script>
```

The default slot is the trigger's content.

### Accessibility

- Panel is `role="alertdialog"` with `aria-modal="true"`, named by the title and
  described by the description (both required).
- Focus starts on **Cancel** (the least-destructive action). **Escape** cancels,
  and a backdrop press cancels by default (opt out with
  `closeOnOutsideClick={false}` for irreversible actions). Focus is trapped and
  body scroll locked; focus returns to the trigger on close.

### Example

```svelte
<AlertDialog
  title="Delete file"
  description="Do you want to delete “report-q3.pdf”? The file will be permanently deleted."
  actionLabel="Delete file"
  cancelLabel="Keep file"
  actionVariant="danger"
  onAction={() => remove()}
>
  Delete file
</AlertDialog>
```

Themeable via `--ds-dialog-*` (shared with Dialog).
