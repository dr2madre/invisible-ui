A carousel with three modes: `slide` (full-bleed background slides with an
overlay), `gallery` (a scrolling row of cards), and `coverflow` — a 3D
"jukebox" where the active item is centered and its neighbors recede (set
`orientation="vertical"` to stack the flow top-to-bottom). Render custom slide
content with the default slot (`let:item let:index let:active`).

### Import

```svelte
<script>
  import Carousel from "@design-system/svelte/Carousel.svelte";
</script>
```

### Props

<PropsTable component="carousel" />

### Accessibility

- Labelled region with previous/next controls and indicator buttons; the active
  slide is tracked for assistive tech.

### Example

```svelte
<Carousel label="Photos" items={[{ image: "/a.jpg", title: "A" }]} />
```

Themeable via `--ds-carousel-*`.
