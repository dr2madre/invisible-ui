A row of overlapping avatars with a "+N" overflow chip when there are more
people than `max`.

### Import

```svelte
<script>
  import AvatarGroup from "@design-system/svelte/AvatarGroup.svelte";
</script>
```

### Props

<PropsTable component="avatar-group" />

### Accessibility

- The group is a labelled list; each avatar exposes its `name` (or `alt`), and
  the overflow chip announces the remaining count.

### Example

```svelte
<AvatarGroup
  label="Project team"
  items={[{ name: "Ada Lovelace" }, { name: "Grace Hopper" }]}
/>
```

Themeable via `--ds-avatar-*`.
