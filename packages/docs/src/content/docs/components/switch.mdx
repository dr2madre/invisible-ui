A switch for instant on/off settings, built on a native
`<input type="checkbox" role="switch">` so screen readers announce on/off and the
value participates in forms.

### Import

```svelte
<script>
  import SwitchDemo from "../../../demos/SwitchDemo.svelte";
</script>
```

### Props

<PropsTable component="switch" />

### Accessibility

- Native `<input type="checkbox" role="switch">`: the browser provides Space
  activation, focus and form participation, and screen readers announce "on"/"off".
- A `label` is required; the native input is wrapped in a `<label>`, so the
  visible text names the control and clicking it toggles the switch.

### Example

```svelte
<Switch label="Wi-Fi" checked onCheckedChange={(on) => setWifi(on)} />
```

Themeable via `--ds-switch-*`.
