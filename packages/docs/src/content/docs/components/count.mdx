A notification number / badge that clamps to `max` ("N+") with an optional dot
mode for presence indicators.

### Import

```svelte
<script>
  import Count from "@design-system/svelte/Count.svelte";
</script>
```

### Props

<PropsTable component="count" />

### Accessibility

- Pass `label` so the number is announced with meaning (e.g. "5 unread messages").

### Example

```svelte
<Count count={128} max={99} label="notifications" />
```

Themeable via `--ds-count-*`.
