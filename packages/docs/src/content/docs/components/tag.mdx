A small status chip: a colored label with optional icons, removable, and able
to host a [Count](/components/count/).

### Import

```svelte
<script>
  import Tag from "@design-system/svelte/Tag.svelte";
</script>
```

### Props

<PropsTable component="tag" />

The default slot is the label; the `icon` slot holds a leading icon and the
`trailing` slot a trailing one.

### Example

```svelte
<Tag status="success" variant="solid">Published</Tag>
<Tag status="info" removable onRemove={() => {}}>Filter</Tag>
<Tag status="danger">
  <Icon slot="icon"><!-- … --></Icon>
  Blocked
</Tag>
```

Themeable via `--ds-tag-*`.
