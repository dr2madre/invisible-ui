A navigation trail from the site root to the current page. Linked ancestors are
underlined; the last item is the current page (`aria-current="page"`, in the
selection color).

### Import

```svelte
<script>
  import Breadcrumb from "@design-system/svelte/Breadcrumb.svelte";
</script>
```

### Props

<PropsTable component="breadcrumb" />

Items are data-driven; mark the first with `home: true` for a leading home glyph,
and omit `href` on the current page.

### Example

```svelte
<Breadcrumb
  items={[
    { label: "Home", href: "/", home: true },
    { label: "Components", href: "/components" },
    { label: "Breadcrumb" },
  ]}
/>
```

Themeable via `--ds-breadcrumb-*`.
