A WAI-ARIA tree with roving tabindex, expand/collapse and keyboard navigation.

### Import

```svelte
<script>
  import TreeView from "@design-system/svelte/TreeView.svelte";
</script>
```

### Props

<PropsTable component="tree-view" />

### Accessibility

- `role="tree"` with roving tabindex; Up/Down move between visible rows,
  Left/Right collapse/expand (or move to parent/child), Home/End jump to ends,
  Enter/Space select.

### Example

```svelte
<TreeView label="Files" nodes={[{ value: "src", children: [{ value: "index.ts" }] }]} expanded={["src"]} />
```

Themeable via `--ds-tree-*`.
