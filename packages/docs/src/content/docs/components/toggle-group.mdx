A set of toggle buttons — `single` (at most one pressed) or `multiple` (any
number) — with roving tabindex.

### Import

```svelte
<script>
  import ToggleGroup from "@design-system/svelte/ToggleGroup.svelte";
</script>
```

### Props

<PropsTable component="toggle-group" />

### Accessibility

- Roving tabindex across the buttons; each exposes `aria-pressed`.

### Example

```svelte
<ToggleGroup type="multiple" label="Format" items={[{ value: "bold", label: "Bold" }]} />
```

Themeable via `--ds-toggle-*`.
