A scroll container with custom overlay scrollbars. Scrolling stays native (the
viewport is a real overflow container); the scrollbars are painted from the
scroll metrics.

### Import

```svelte
<script>
  import ScrollArea from "@design-system/svelte/ScrollArea.svelte";
</script>
```

### Props

<PropsTable component="scroll-area" />

The scrollable content is the default slot.

### Accessibility

- Native scrolling (keyboard works); custom scrollbars are decorative.

### Example

```svelte
<ScrollArea maxHeight="10rem" label="Terms"><p>…</p></ScrollArea>
```

Themeable via `--ds-scroll-area-*`.
