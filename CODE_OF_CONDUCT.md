# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful and constructive. Harassment or exclusionary behaviour
is not tolerated. Report concerns to the maintainers at the contact in
`SECURITY.md`. Maintainers may remove, edit, or reject contributions and
comments that violate this code.
